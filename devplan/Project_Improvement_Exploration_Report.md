### 🟡 중요 (P2)

#### [P2-1] 추가 언어 UI 지원 (일본어, 중국어)

| 항목 | 내용 |
|:---|:---|
| **ID** | `I18N-001` |
| **카테고리** | 🌐 i18n |
| **복잡도** | Low |
| **대상 파일** | `apps/desktop/src/i18n/locales/ja.ts`, `apps/desktop/src/i18n/locales/zh.ts`, `apps/desktop/src/i18n/types.ts` |

**현재 상태:** v0.1.3에서 한국어/영어 UI가 구현되었으나, 글로벌 사용자를 위한 일본어, 중국어 지원이 부족합니다. `SupportedLocale` 타입이 `"ko" | "en"`으로만 정의되어 있습니다.

**개선 내용:**
1. `i18n/locales/ja.ts` 파일 생성 (일본어 번역)
2. `i18n/locales/zh.ts` 파일 생성 (중국어 간체)
3. `SupportedLocale` 타입에 `'ja' | 'zh'` 추가
4. 언어 선택 드롭다운에 새 언어 옵션 추가

**기대 효과:**
- 아시아 지역 사용자 접근성 향상
- 모드 번역 프로젝트답게 다양한 언어 지원

|:---|:---|
| **ID** | `DOC-001` |

**개선 내용:**

2. `core/Cargo.toml`에 문서화 메타데이터 추가
3. GitHub Actions 워크플로우로 자동 문서 배포
4. README에 문서 링크 추가

**기대 효과:**
- 개발자 진입 장벽 감소
- 문서화 점수 80 → 88+ 예상
