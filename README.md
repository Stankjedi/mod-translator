# Mod Translator

스팀 창작마당(Steam Workshop) 모드를 자동으로 한국어로 번역해주는 Windows 데스크톱 프로그램입니다.

## �� 소개

Mod Translator는 스팀 게임의 모드 파일을 자동으로 번역하여 게임을 더 편하게 즐길 수 있도록 도와주는 프로그램입니다. 복잡한 설정 없이 몇 번의 클릭만으로 구독한 모드들을 한국어로 번역할 수 있습니다.

### 특징
- ✅ **안전한 번역**: 코드와 XML 태그는 그대로 유지하고 텍스트만 번역합니다
- 🔍 **자동 감지**: 스팀 설치 경로와 구독 모드를 자동으로 찾아줍니다
- 💾 **백업 기능**: 원본 파일을 자동으로 백업하여 언제든지 되돌릴 수 있습니다
- 🤖 **다양한 AI 지원**: 여러 AI 번역 모델 중 선택하여 사용할 수 있습니다
- 🌍 **다국어 지원**: 영어, 중국어 등 다양한 언어에서 한국어로 번역 가능합니다

## 💻 시스템 요구사항

- **운영체제**: Windows 10 또는 Windows 11 (64비트)
- **디스크 공간**: 최소 1GB 이상의 여유 공간
- **기타**: 별도의 .NET이나 VC++ 재배포 패키지 설치 불필요

## 📥 다운로드 및 설치

1. [Releases 페이지](../../releases)에서 최신 버전의 `mod-translator-setup.exe` 파일을 다운로드하세요
2. 다운로드한 설치 파일을 실행하세요
3. 설치 마법사의 안내에 따라 설치를 완료하세요
4. 설치가 완료되면 바탕화면 또는 시작 메뉴에서 Mod Translator를 실행하세요

> **참고**: 아직 정식 버전이 출시되지 않은 경우 "Pre-release" 버전을 사용하실 수 있습니다.

## 🚀 사용 방법

### 1단계: 스팀 경로 설정
프로그램을 처음 실행하면 대시보드 화면이 나타납니다.
- "스팀 경로 자동 감지" 버튼을 클릭하세요
- 프로그램이 자동으로 스팀 설치 위치를 찾아줍니다
- 자동 감지가 안 되는 경우, 수동으로 스팀 폴더를 선택할 수 있습니다

### 2단계: 모드 스캔
- 스팀 경로가 설정되면 "모드 스캔" 버튼을 클릭하세요
- 구독 중인 모든 창작마당 모드가 목록에 표시됩니다
- 각 모드의 크기와 파일 개수를 확인할 수 있습니다

### 3단계: 번역할 모드 선택
- 목록에서 번역하고 싶은 모드를 선택하세요 (여러 개 선택 가능)
- 각 모드 옆의 체크박스를 클릭하여 선택합니다
- 선택한 모드의 총 용량과 예상 시간이 표시됩니다

### 4단계: 번역 설정
설정 화면에서 다음 옵션들을 조정할 수 있습니다:
- **AI 모델 선택**: 사용할 번역 AI 모델을 선택하세요
- **API 키 입력**: 선택한 AI 서비스의 API 키를 입력하세요
- **번역 품질**: 빠른 번역 또는 고품질 번역 중 선택하세요

### 5단계: 번역 시작
- "번역 시작" 버튼을 클릭하세요
- 진행 상황 화면에서 번역 진행률을 실시간으로 확인할 수 있습니다
- 번역 중에는 각 파일의 처리 상태를 볼 수 있습니다

### 6단계: 결과 확인 및 적용
- 번역이 완료되면 결과 미리보기가 표시됩니다
- 번역 결과를 검토한 후 "적용" 버튼을 클릭하세요
- 적용하면 게임에서 즉시 번역된 내용을 확인할 수 있습니다

## 💡 주요 기능 상세 설명

### 자동 백업
- 번역을 적용하기 전에 원본 파일이 자동으로 백업됩니다
- 문제가 발생하면 언제든지 원본 상태로 되돌릴 수 있습니다
- 백업 파일은 프로그램 데이터 폴더에 안전하게 보관됩니다

### 안전한 번역
- XML, JSON, Lua 등의 코드 구조는 그대로 유지됩니다
- 변수명, 태그, 특수 문자는 번역하지 않습니다
- 번역 후 자동 검증을 통해 파일 손상을 방지합니다
- 검증에 실패하면 자동으로 원본으로 복구됩니다

### 번역 미리보기
- 실제 적용하기 전에 번역 결과를 미리 볼 수 있습니다
- 원본과 번역본을 나란히 비교할 수 있습니다
- 마음에 들지 않는 부분은 수동으로 수정할 수 있습니다

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 번역하면 모드가 깨지거나 게임이 실행되지 않을까요?
**A**: 걱정하지 마세요! Mod Translator는 다음과 같은 안전 장치를 갖추고 있습니다:
- 코드와 XML 태그는 절대 번역하지 않습니다
- 번역 후 자동으로 파일 구조를 검증합니다
- 문제가 감지되면 자동으로 롤백(복구)합니다
- 모든 원본 파일은 안전하게 백업됩니다

### Q2. 어떤 게임의 모드를 번역할 수 있나요?
**A**: 스팀 창작마당을 사용하는 모든 게임의 모드를 번역할 수 있습니다. 대표적인 예시:
- Cities: Skylines
- RimWorld
- Stellaris
- Europa Universalis IV
- Hearts of Iron IV
- 기타 창작마당을 지원하는 모든 게임

### Q3. 번역에 비용이 드나요?
**A**: 프로그램 자체는 무료이지만, AI 번역 서비스 이용에는 비용이 발생할 수 있습니다:
- 각 AI 서비스(Google Gemini, OpenAI GPT 등)의 API 키가 필요합니다
- API 사용량에 따라 해당 서비스의 요금이 부과됩니다
- 대부분의 서비스는 소량의 무료 사용량을 제공합니다

### Q4. 자동 감지가 되지 않아요
**A**: 다음 방법을 시도해보세요:
1. 설정 메뉴에서 "스팀 경로 수동 설정"을 선택하세요
2. 일반적인 스팀 설치 경로: `C:\Program Files (x86)\Steam`
3. 그래도 안 되면 스팀 라이브러리 폴더를 직접 선택하세요

### Q5. 특정 모드만 번역되지 않아요
**A**: 몇 가지 가능한 원인이 있습니다:
- 모드 파일이 비표준 형식을 사용할 수 있습니다
- 진행 화면에서 "파일 수동 선택" 옵션을 사용해보세요
- 그래도 안 되면 Issue를 등록하여 문의해주세요

### Q6. 번역을 원래대로 되돌리고 싶어요
**A**: 다음 두 가지 방법이 있습니다:
1. 프로그램에서 해당 모드를 선택하고 "원본 복구" 버튼 클릭
2. 스팀 창작마당에서 해당 모드를 구독 취소 후 다시 구독

### Q7. 번역된 내용이 부자연스러워요
**A**: 번역 품질을 높이려면:
- 설정에서 더 좋은 AI 모델을 선택하세요 (예: GPT-4)
- "고품질 번역" 옵션을 활성화하세요
- 번역 후 미리보기에서 직접 수정할 수 있습니다

### Q8. 여러 개의 모드를 한 번에 번역할 수 있나요?
**A**: 네! 모드 목록에서 여러 모드를 선택하고 한 번에 번역을 시작할 수 있습니다. 프로그램이 순차적으로 처리합니다.

## 🔒 개인정보 및 보안

- **데이터 전송**: 번역 요청에 필요한 텍스트만 AI 서비스로 전송됩니다

## 🛠️ 문제 해결

### 프로그램이 실행되지 않아요
1. Windows 10 또는 11인지 확인하세요
2. 64비트 운영체제인지 확인하세요
3. 프로그램을 관리자 권한으로 실행해보세요
4. 최신 버전을 다시 다운로드하여 설치해보세요

### 번역이 너무 느려요
1. 설정에서 "빠른 번역" 옵션을 활성화하세요
2. 동시 처리 개수를 늘려보세요 (설정 > 고급 옵션)
3. 더 빠른 AI 모델을 선택하세요
4. 인터넷 연결 상태를 확인하세요

### 번역 후 게임이 실행되지 않아요
1. 즉시 "원본 복구" 기능을 사용하세요
2. 해당 모드를 스팀에서 구독 취소 후 재구독하세요
3. 게임 파일 무결성 검사를 실행하세요 (스팀 > 속성 > 파일)
4. Issue를 등록하여 문제를 보고해주세요

### API 키 오류가 발생해요
1. API 키를 정확히 복사했는지 확인하세요 (공백 없이)
2. 해당 AI 서비스의 계정 상태를 확인하세요
3. 사용 한도를 초과하지 않았는지 확인하세요
4. 인터넷 연결 상태를 확인하세요
