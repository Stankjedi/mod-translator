Mod Translator (Windows)

여러 스팀 창작마당 모드를 자동으로 한국어로 번역하는 데스크톱 앱. 번역 시 코드·태그는 보존하고 텍스트만 치환합니다.

다운로드

최신 설치 파일: Releases 탭에서 mod-translator-setup.exe 받기
(배포 전이라면 “Pre-release” 자산을 사용)

주요 기능

스팀 설치 경로 자동 감지, 구독 모드 자동 스캔

영어·중국어 등 → 한국어 번역

코드/태그 보존, 번역 후 검수 및 롤백

파일 백업과 원본명 유지, 적용 자동화

다중 AI 모델 지원(선택식)

시스템 요구 사항

Windows 10/11 64-bit

디스크 여유 공간: 1GB 이상

.NET/VC++ 재배포 패키지 불필요(포함된 Tauri 런타임 사용)

빠른 시작

설치 파일 실행 → 앱 시작

첫 화면에서 스팀 경로 자동 감지 클릭

모드 스캔 후 목록에서 번역할 모드 선택

번역 옵션 선택 후 번역 시작

결과 검토 → 적용을 누르면 게임에서 즉시 반영

스크린샷

자주 묻는 질문

Q. 코드나 XML 태그가 번역되어 모드가 깨질까 걱정됩니다.
A. 파서가 코드·태그를 잠그고 텍스트 노드만 번역합니다. 사후 검증에서 변형이 감지되면 자동 롤백합니다. 자세한 기준은 TRANSLATION.md
 참고.

Q. 특정 모드에서 자동 탐지가 안 됩니다.
A. 진행 화면의 파일 선택에서 번역 대상 파일을 수동 선택하세요. 문제 해결은 TROUBLESHOOTING
.

개인정보·네트워크

번역 요청 외 데이터는 외부로 전송하지 않습니다.

로그는 로컬에만 저장. 전송 기능 없음.
